# Video Chat with WebRTC and Firebase
WebRTC project highly inspirated by https://github.com/fireship-io/webrtc-firebase-demo.

## Usage
To run locally:
```
npm install

npm run dev
```

To deploy on Heroku:

```
heroku init

git add .
git commit -m "Init project"
git push heroku main
```
